Archive Prep Tools

A small suite for preparing audiovisual and tabular archive materials.
This page is served by your own computer. To use the web tools below, the local server must be running — see the README for one-time setup and the ./start.sh command. The first audio/video job downloads a transcription model once, then runs quickly.
Web tool

AV File Access Preparation

Point to a folder of audio/video files and a metadata CSV. The tool transcribes speech (SRT + RTF), writes a short content description per file, pulls out names, places, and music/poem/book titles, and builds one enriched metadata spreadsheet — all on your machine.
Open tool →
Web tool

Data-Prep Tool

Upload a CSV or point to a Google Sheet. The tool checks every column for quality problems (empty/outlier/inconsistent values), plans duplicate merges, and generates a README and data dictionary — leaving your original file untouched.
Open tool →
Command line

Merritt Harvester

A Terminal script that pulls collection metadata from a private Merritt collection into CSV/JSON. This one has no web form — you run it from Terminal with a session cookie copied from your browser. Step-by-step instructions are in the README.
